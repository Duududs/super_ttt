## ✩ Funcionalidades

#### ⊰ Tela Inicial

- Escolha do modo de jogo:
  - **Local vs Local**
  - **Local vs Robô**
- Interface inspirada em menus de jogos clássicos

#### ⊰ Jogo

- Tabuleiro **3x3**
- Placar em tempo real:
  - **Mario vs Luigi** ou **Mario vs Robô**
- Detecção automática de:
  - Vitória
  - Empate
- Botão de **reiniciar partida**

#### ⊰ Modo Robô

IA simples, porém estratégica:

- Tenta **ganhar** quando possível
- **Bloqueia** jogadas do jogador
- Prioriza:
  - Centro
  - Cantos
  - Jogadas inteligentes básicas

#### ⊰ Visual & UX

- Estilo **pixel art**
- Tema Mario (cores, ícones, blocos, fontes)
- Animações sutis e feedback visual
- Totalmente responsivo

#### ⊰ Navegação

- Roteamento básico
- Página principal
- Fallback **404**

#### ⊰ Como Rodar Localmente

```
Clone o repositório
git clone https://github.com/Duududs/super_ttt.git

Entre no projeto
cd super_ttt

Instale as dependências
npm install

Inicie o servidor
npm run dev
```
